# Mi Gestor de Notas

Buenas, este es mi proyecto para Entornos de Desarrollo. Es una aplicación bastante simple en Python para guardar notas y que no se te olviden las cosas.

### ¿Qué hace mi programa?
* **Añadir:** Escribes algo y se guarda en la lista.
* **Ver:** Te enseña todas las notas que has escrito con un numerito al lado.
* **Borrar:** Si ya no quieres una nota, pones su número y la quita.
* **Salir:** Pues eso, cierra el programa cuando terminas.

### Cómo hacerlo funcionar
1. Necesitas tener **Python** instalado en el PC.
2. Abres la carpeta en la terminal.
3. Escribes `python nombre_de_tu_archivo.py` y ya puedes empezar a usarlo.
